Hello!

網頁程式
原始碼

當網頁被點擊或是載入後超過三秒，就會從資料庫中隨機選擇一個網址，並自動導向過去。另外首頁的文章也是隨機從資料庫讀取一篇的，除了幾篇短文以外，其他皆是由文章由好用工具箱產出。

這是我第一次設計網頁。裡面的許多功能都是從彭彭老師的教學影片、Bootstrap (六角學院譯)以及其他網路上學習到並嘗試加入進來的。像是隨機取得首頁文章的 AJAX、突如其來的自動轉址、操作 BOM 來顯示現在時間等等。讓我印象最深刻的是 Bootstrap，裡面包含了許多常用的元件，讓剛接觸網頁設計還不太會使用 CSS 的我也能快速的做出一個漂亮的網頁。

關於自動轉址的部分，靈感來自「統神端火鍋」的各種釣魚影片，類似的應用有短網址服務，但這種技術還是存在一些隱憂，像是可能被有心人士拿去製作詐騙網站、下載電腦病毒等等，也可能會利用預覽標題和說明來散布假新聞。

The 時鐘

網頁程式
原始碼

教室裡沒有時鐘，無法知道時間？那就自己來做一個吧!

透過 JavaScript 的 Date() 取得時間，再利用 canvas 來繪製時鐘。

不過目前CSS還沒有很熟悉，所以網頁大小是設定1920×1080，不會隨著視窗而改變，也沒有RWD設計。

這是我初次嘗試使用 canvas 來繪圖，因此只有使用 JavaScript 原生的API來操作，雖然也有像是 jCanvas 這樣基於 jQuery 的 library 可以來操作。在製作程式的過程中，我覺得就像在教電腦使操作小畫家來畫圖一樣，要使用什麼模式、從哪裡下筆、要畫到哪裡、要用什麼顏色、線條要多粗…等等，十分有趣。

DVD動畫

網頁程式
原始碼

小時候總是在電視螢幕上看到 DVD 的圖案在上面移動，碰到邊框時反彈並更換顏色，每次都期待著撞到四個角落反彈的時候。

這次利用 JavaScript 以及 CSS 來製作出動畫的效果。撞到邊界時會切換成一種與背景(黑色)不相近的顏色。

雖然有設計指令可以設定內容，以及 cookie 可以存取設定，不過因為不會只製作設定選單 GUI，只能透過 console 輸入，功能不是很完善。

16減9

網頁程式
原始碼

在網路上看到了一個阿嬤算數學的影片。她在嘗試計算16-9的答案，不過算了很久很久，用了好幾張紙都還沒算出來，於是我就利用電腦來幫忙運算，藉此減輕那位阿嬤的負擔。

在過程中利用遞迴的方式形成無限迴圈，使用 setTimeout() 函式來製造停頓，以及 flex 來使算式由左至右排列且溢出則自動換行。
